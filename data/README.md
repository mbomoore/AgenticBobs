# Data

This directory contains data files, samples, and test assets.

## Directory Structure

### `samples/`

Sample data files and example assets:

- `sample.txt` - Sample business process description text
- `generated.bpmn` - Generated BPMN process example (spec transfer process)

## Usage

These files are used by:

- Tests and validation
- Example workflows
- Demonstration notebooks
- Development and experimentation

## File Formats

- `.txt` - Text descriptions and sample content
- `.bpmn` - Business Process Model and Notation XML files
- `.dmn` - Decision Model and Notation files (when present)
- `.xes` - Event log files (when present)
